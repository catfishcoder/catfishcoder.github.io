Markdown语法整理

2021.12.23

文中语法为Markdown的语法，展示效果为实际输出。

一. 标题

语法：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

二. 分级目录

语法：

+ 总目录
  + 一级目录
    + 二级目录
      + 三级目录
        + ......

说明：层级结构实现方式为 +/* 后面跟一个空格。一般支持三级列表，第一级实心圆点，第二级空心圆点，第三级及以后为实心方块.要实现层级列表，新的一级要比上一级多一个Tab键。两次回车离开本级目录，方向下键离开层级列表。

展示效果：

  • 总目录
    • 一级目录
      • 二级目录
        • 三级目录
          • ……

三.显示效果

语法：

*斜体*; **粗体**; ***粗斜体***	; ~小字体~; ~~删除线~~

<u>下划线\</u>; ==高亮==; 上标^1^; 下标~1~

文本[^脚注]; [^脚注]: 描述; --/**** 分割线

展示效果：

斜体; 粗体; 粗斜体 ; ~小字体~; 删除线

下划线 ; ==高亮==; 上标^x^; 下标~x~; 文本1

说明：脚注的作用是在文本中添加标记，在在章节末尾通过标记添加具体描述,实现在文本中鼠标悬停查看描述

四. 链接

Markdown有行内式、参考式、自动链接、锚点四种链接形式

  1. 行内式：

    语法：

    [这是百度](https://www.baidu.com)
[这是Google](https://www.google.com "打开百度")

    `` 展示效果：

    这是百度 这是Google

    说明：方括号里是链接文字，括号内写链接地址，括号内的双引号字段表示链接属性，鼠标悬停会显示链接属性；

  2. 参考式

    语法：

    [baidu][1] 没有 [Google][2] 牛皮

[1]:https://www.baidu.com "open Google"
[2]:https://www.google.com "open baidu"

    展示效果：

    baidu 没有 Google 牛皮

    说明：注意链接地址和链接属性之间的空格；

  3. 自动链接

    语法：

    <https://www.baidu.com>
<https://www.google.com>

    展示效果：

    https://www.baidu.com

    https://www.google.com

    说明：Markdow支n持以比较简短的自动链接形式来处理网址和电子邮件信箱，只要是用<>括起来，Markdown就会自动把它转换成链接；

  4. 锚点

    语法：

    <a id= "1" >标题</a>
	  示例内容1...
	  示例内容2...
[锚点](#1)

    展示效果：

    标题 示例内容1… 示例内容2… 锚点

    说明：Markdown中的锚点要配合HTML使用，在需要设置锚点的地方插入标记，在其他地方利用 # 链接到锚点

五.列表

  1. 无序列表

    语法：

    + 无序列表1；
+ 无序列表2；
+ 无序列表3；

    `` 展示效果：

    • 无序列表1；
    • 无序列表2；
    • 无序列表3；

    说明：使用 + /* /- 表示无序列表，注意空格；

  2. 有序列表

    语法：

     1. 有序列表1；
 2. 有序列表2；
 3. 有序列表3；

    展示效果：

    1. 有序列表1；
    2. 有序列表2;
    3. 有序列表3;

  3. 任务列表

    语法：

     + [ ] 任务列表1；
 + [x] 任务列表2；

    `` 展示效果：

    • 任务列表1；
    • 任务列表2；

说明：方括号里是空格表示未完成，x表示已完成

六.表格

名字 性别 年龄
张三 男 22

王五 男 22

七. 代码块

  1. 栅栏代码

    语法：

    \```编程语言
	中间是代码块
\```

    ``

  2. 引用代码

    语法：

    ' hello world '

    展示效果： hello world!

    说明：反引号括起来的是引用代码，markdown中反引号不能转移用单引号表示；

  3. 引用文字

    语法：

    > 一级引用
  >> 二级引用
    >> 三级引用

    `` 展示效果：

    一级引用

    二级引用

    三级引用
